# jlox: A Lox Interpreter written in Java

A Java interpreter for Lox, the programming language from Crafting Interpreters by Robert Nystorm.

## Run Demo
1. `git clone https://github.com/jacobgoldfarb/jlox`
2. `cd {repo-directory}`
3. `make run_demo`

## Run REPL
1. `git clone https://github.com/jacobgoldfarb/jlox`
2. `cd {repo-directory}`
3. `make crun`
